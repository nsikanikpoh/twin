You enter a URL in a browser and press return; describe what happens technically in as much details as possible.

When you enter in a url in the browser and hit enter, The browser checks in the DNS cache to see if it finds an IP address attached to that url, if it doesn't find,
then your ISP's DNS is going to issue a DNS query to search for an IP address attached to that url and When it finds it, then, the browser will initiate a tcp
 connection with the server that is associated with that IP, and when that happens, your browser will send http request, the server will respond with an http response, your the browser will process it and render a webpage.
